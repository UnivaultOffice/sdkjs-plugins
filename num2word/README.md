## Overview

This plugins shows hot to how to convert a number to a word (a number written as a string).

It is called "num2word" in the interface and isn't installed by default in cloud, [self-hosted](https://github.com/UnivaultOffice/DocumentServer) and [desktop version](https://github.com/UnivaultOffice/DesktopEditors) of UNIVAULTOFFICE editors. 

## How to use

1. Select any number.
2. Open the Plugins tab and press "num2word".
3. The number converted to a string in Russian will be inserted in the cell below.
